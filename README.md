# TikTok-Parser

## Install:
> pip install -U selenium

## Run:
> python src/main.py --tag "TAGNAME" --minNFollowers N --maxNFollowers M --minNLikes J --maxNLikes K

For example:
> python src/main.py --tag magic,foot --minNFollowers 100000 --maxNFollowers 10000000000  --minNLikes 1 --maxNLikes 10000000000
